First, run "DDBDM.m" to get the BER_sweep data for DDBDM. You can modify the parameters of the delay-doppler grid, the LEO channel, or other parameters related to the number of points in the BER, SNR, and CNPR.
Then, run "ofdm_code.m" to get the BER_sweep for the comparative modulations complemented by the OFDM protocol. Make sure both parameters for the channel coincide.
Run "grafica.m" to see the final results.
